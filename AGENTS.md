# AGENTS.md

## Projet
`pong_devastator_solo` est un mini-projet d’apprentissage de Python avec `pygame-ce`.

## Environnement
- Linux
- VSCode
- Profil VSCode : `Python_PyGame_CE`
- Environnement Python : `.venv`
- Python : `3.12.3`
- Bibliothèque : `pygame-ce`

## Règles
- Garder le projet simple, progressif et pédagogique.
- Éviter la surconception.
- Pour l’instant, travailler principalement dans `main.py`.
- Ne pas créer de nouveaux fichiers ou dossiers sans besoin réel.
- Faire des changements petits, lisibles et faciles à tester.

## Style
- Commentaires en français.
- Noms Python en français sans accents.
- Texte utilisateur en français.
- Conserver en anglais les éléments imposés par Python et Pygame.

## Objectif actuel
Construire un Pong solo très simple, inspiré de *Code the Classics* volume 1, puis l’enrichir par petites étapes.
